# Vue 开发记录

## 项目建立

### 1.创建一个项目

使用Vue CLI 3

```bash
vue create hello-world
```

图形化界面创建和管理项目

```bash
vue ui
```

使用Vue CLI 2

```bash
npm install -g @vue/cli-init
```

### 2.安装使用Vuetify

```bash
vue add vuetify
```

### 3.安装Vue Router

```sh
vue add router
```
